
# Add Product UI Mockup (Medium-Fidelity)

This document describes the UI/UX flow for adding a product in the admin dashboard.  
The mockups are divided into 4 steps:

## 1. Step 1 – Select Category
- Dropdown to select category
- Attributes load dynamically based on selected category

## 2. Step 2 – Product Information
- Product name
- Slug
- Base price
- Description (short)
- Upload images
- Active / Featured toggles

## 3. Step 3 – Attributes & Values
- Load attributes based on the selected category
- Non-variant attributes → single select
- Variant attributes → multi-select
- Table format for clean UX

## 4. Step 4 – Variants (Auto Generated)
- Generated variant list based on selected variant attributes
- Editable SKU, price, stock, image
- Optional: add/remove variants manually

The corresponding image mockup will include:
- 4 sections in a single composite image
- Medium-fidelity layout (desktop 1920×1080)
- Clean admin-style UI
